# house-price-prediction
My first machine learning project - House Price Prediction using Scikit-Learn.

🏠 House Price Prediction (My First ML Project)

Hi! 👋
This is my first GitHub repository.
In this project, I built a Machine Learning model that predicts house prices using different regression algorithms and proper data preprocessing techniques.
This project helped me understand how real-world ML workflows are structured.

📌 What This Project Does

1) Loads housing dataset (train.csv)

2) Splits data into training and testing sets

3) Handles missing values

4) Encodes categorical features

5) Scales numerical features

6) Trains regression models

7) Evaluates model performance

8) Checks for overfitting

🧠 Concepts I Learned
While building this project, I learned:

✅ Train-test splitting

✅ Feature preprocessing using Pipelines

✅ ColumnTransformer in Scikit-Learn

✅ One-Hot Encoding

✅ Feature Scaling

✅ Model evaluation (R², MAE, RMSE)

✅ Overfitting detection

🤖 Models Implemented

1) Linear Regression

2) Decision Tree Regressor

3) Random Forest Regressor (Final Model Used)

📊 Evaluation Metrics Used

1) R² Score

2) Mean Absolute Error (MAE)

3) Root Mean Squared Error (RMSE)

4) Training vs Testing Score comparison

🛠️ Technologies Used
